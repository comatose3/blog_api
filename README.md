# Blog API

Backend-приложение для блог-платформы, реализованное на Java с использованием Spring Boot.  
Проект представляет собой REST API для работы с постами, комментариями и пользователями, а также демонстрирует событийное взаимодействие через Apache Kafka.

## Стек технологий

- **Java**
- **Spring Boot**
- **PostgreSQL**
- **Apache Kafka**
- **Docker / Docker Compose**
- **Git**

## Функциональность

- CRUD-операции для постов
- Работа с комментариями
- Хранение данных в PostgreSQL
- Асинхронная обработка событий через Kafka  
  (например, события создания постов или комментариев)
- REST API
- Контейнеризированный запуск приложения и инфраструктуры

## Цели проекта

- Практика backend-разработки на Java
- Изучение Spring Boot и REST-архитектуры
- Работа с реляционной базой данных
- Использование Kafka для событийного взаимодействия
- Освоение Docker и docker-compose
- Ведение проекта с использованием Git

## Архитектура

Проект построен по классической layered-архитектуре:

- **Controller layer** — обработка HTTP-запросов
- **Service layer** — бизнес-логика
- **Repository layer** — работа с базой данных
- **Kafka producer / consumer** — публикация и обработка событий

Такой подход позволяет разделить ответственность компонентов и упростить расширение проекта.
